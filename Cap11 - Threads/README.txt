1. Troque de diretório: no CMD, digite -> cd diretório\até\src
2. Compile os arquivos Java: agora digite -> javac caso1\*java caso2\*java caso3\*java caso4\*java caso5\*java caso6\*java caso7\*java caso8\*java
3. Execute a classe principal de cada pacote para ver cada exemplo:
a. digite -> java caso1.CriandoThreads1
b. digite -> java caso2.CriandoThreads2
c. digite -> java caso3.CriandoThreads3
d. digite -> java caso4.CriandoThreads4
e. digite -> java caso5.CriandoThreads5
f. digite -> java caso6.CriandoThreads6
g. digite -> java caso7.CriandoThreads7
h. digite -> java caso8.CriandoThreads8
